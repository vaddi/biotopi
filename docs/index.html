BiotoPi A simple Raspberry Pi controler for Habitats

Biotopi is a small Software to use a Raspberry Pi as a Webserver to controll Terrariums and Aquariums via Webinterface. Basicly it is a PHP API, which can be extdended by simple adding new PHP files under the folder `inc/class/controllers/`.
